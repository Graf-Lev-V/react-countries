# First React Practice Project

Site: [https://github.com/Graf-Lev-V/react-countries](https://graf-lev-v.github.io/react-countries/)

This is my first React project, built to practice the core React concepts from scratch.

In this project, I learned and practiced:
- components, JSX, props
- rendering lists with `map` and `key`
- `useState`
- controlled inputs
- events like `onChange` and `onSubmit`
- conditional rendering
- lifting state up
- `useEffect`, `useRef`
- a custom `useFetch` hook
- working with APIs using `fetch`
- `async/await`, `try/catch/finally`
- `AbortController`
- React Router (`HashRouter`, `Routes`, `Route`, `Navigate`, `Outlet`)
- dynamic routes like `/users/:id`
- forms, validation, and prefilling data from an API

## Features
- home page
- user page by id
- navigation between users
- profile form with validation
- 404 page

## Stack
React, Vite, React Router, JavaScript, CSS

## API
This project uses JSONPlaceholder.
